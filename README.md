# cartpilot# CartPilot v2 - AI-Powered Ecommerce Optimization

An AI tool that reviews data from Shopify and Klaviyo, suggests optimization changes, and pushes approved changes directly back to your ecommerce platforms.

## Architecture
Loveable (Frontend)
↓
Supabase (Database + Auto-generated REST API)
↓
N8N Cloud (Workflows & Automation)
↓
ChatGPT (Suggestion Engine)
↓
Shopify + Klaviyo APIsLoveable (Frontend)
↓
Supabase (Database + Auto-generated REST API)
↓
N8N Cloud (Workflows & Automation)
↓
ChatGPT (Suggestion Engine)
↓
Shopify + Klaviyo APIs
## Tech Stack

- **Frontend**: Loveable (React-based)
- **Backend**: Supabase (PostgreSQL + Auto REST API)
- **Workflows**: N8N Cloud
- **AI**: OpenAI GPT-4
- **Integrations**: Shopify, Klaviyo
